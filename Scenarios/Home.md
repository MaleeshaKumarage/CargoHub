# Home / Landing

## Signed-in user opens home

If the user is signed in and opens the home (root) page, then the app may redirect them to the dashboard or show a welcome message with links to the main areas (bookings, create booking, etc.). Behaviour depends on implementation.

## Not signed in

If the user is not signed in and opens the home page, then the app may show a landing or welcome page with links to Login and Register. Clicking Login or Register takes them to the corresponding page.
